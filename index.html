Digital Communications

March-April 2019

A robust system in Python that can transmit text files between two computers using only speaker and microphone. Implemented with Huffman encoding for compression, Berlekamp-Welch for error correction, and frequency modulation/DFT analysis for transmitting and receiving signals. Achieved a final transfer rate of 100 bits/second. Developed alongside Amal Mehta, Wesley Wang, and Srikar Boinapally.
